# ホットクック メニューブラウザ

シャープ ホットクックの全メニューを写真付きでブラウズ／検索できる、個人用の静的ビューア（PWA）。
スマホのホーム画面に追加すると、オフラインでも使えます。

- 料理名・番号で検索
- 系統（煮物・カレー・スープ・スイーツ…）や主材料（鶏・豚・牛・魚介…）で絞り込み
- サイズ・調理時間でソート、系統ごとの見出し表示
- 写真は COCORO KITCHEN の画像を参照（読めない場合は番号タイル表示）

## 使い方
このページ（GitHub Pages）を開いて「ホーム画面に追加」するだけ。
- iPhone: Safari で開く →共有 →「ホーム画面に追加」
- Android: Chrome で開く →メニュー →「アプリをインストール」
